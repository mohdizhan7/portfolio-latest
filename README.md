# Mohammed Izhan's Portfolio

A clean, high-performance portfolio showcasing 6+ years of supply chain systems design and operations expertise.

## Files

- `index.html` — Main homepage (Work + Projects + Contact)
- `about.html` — About page (Experience + Skills + Education)
- `vercel.json` — Vercel deployment config

## Local Development

Simply open `index.html` in your browser. No build process required.

## Deployment to Vercel

1. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "Update portfolio with latest changes"
   git push origin main
   ```

2. **Deploy to Vercel:**
   - Go to https://vercel.com
   - Import the GitHub repository
   - Vercel will auto-detect and deploy

3. **Custom Domain:**
   - Add your domain in Vercel project settings
   - Update DNS records as instructed
